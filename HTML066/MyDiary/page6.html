ประสบการณ์การเริ่ม Coding ครั้งแรก

9/10/2019

ทุกอย่างเริ่มมาจากอาจารย์ในสาขาส่งใบสมัครเข้าแข่งควบคุมแขนกล Dobot ช่วงเดือนสิงหามาให้ปีหนึ่งไปลงแข่ง ผมเลยรับใบสมัครมาเพื่อไปลงแข่ง โดยที่ผมไม่เคยเขียนโปรแกรมอะไรมาก่อน แต่โชคดีที่การแข่งใช้ code blocky ผลเลยเริ่มฝึกจาก 0 มาทั้งยังงั้นแรกๆก็ยัง งงเรื่อง logic แต่ก็พอจะเข้าได้บ้าง จนถึงวันที่ต้องเดินทางไปสถานที่แข่งขันเพื่อรับหัวข้อที่จะใช้แข่งขัน หลังจากได้รับหัวข้อวันต่อมา(7/10/2019) ก็เป็นวันแข่งขันรอบคัดเลือก วันนั้นตื่นมาก็ปวดท้องคงเพราะตื่นเต้น จากนั้นก็เดินทางไปสถานที่แข่งรอบคัดเลือกที่มหาวิทยาลัยราชมงคลธัญบุรี เริ่มแข่งในช่วงบ่ายใช้เวลาแข่ง 3 ชั่วโมง และประกาศผลรอบคัดเลือกในช่วงเย็นวันนั้น ผลออกมาคือได้เข้าแข่งต่อในรอบ 20 ทีมสุดท้าย

วันแข่งรอบสุดท้ายคือวันที่ 9/10/2019 ที่ไบเทคบางนา วันนั้นคนเยอะมาก เพราะไม่ได้มีแข่งแค่รายการนี้ และก็มีจัดนิทรรศการเทคโนโลยีเยอะแยะไปหมด พอถึงเวลาแข่งก็เกร็จอยู่หน่อยๆเพราะไม่เคยคิดว่าจะมาถึงรอบนี้ ยิ่งเกิดเหตุไม่คาดฝันตอน 30 นาทีสุดท้าย โน๊ตบุ๊คที่เอาไปใช้แข่งจู่ๆก็จอฟ้าขึ้นมา เลยต้องใช้เวลา Restart พักหนึ่งระหว่างนั้นก็นั่งกุมขมับ พอเปิดเครื่องติดก็รีบทำต่อจากที่เซฟไว้และพยายามทำให้ไวที่สุด และส่งโค้ดก่อนหมดเวลา 30 วินาทีสุดท้าย เพื่อหวังคะแนนเวลา หลังจากตรวจการทำงานกรรมการก็ให้ออกไปรอข้างนอก ผ่านไประยะเวลาหนึ่งก็เรียกทุกคนเพื่อประกาศผล และได้รางวัลชนะเลิศอันดับหนึ่งมา

หลังจากนั้นก็เริ่มศึกษาภาษาซี ชนิดที่แค่เอามาใช้งานได้และก็ฝึกเรื่อยๆจนทุกวันนี้
